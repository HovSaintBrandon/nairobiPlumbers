## Recommended Image Dimensions
- **Hero images**: 1920×1080px (landscape)
- **Product images**: 800×600px or 4:3 ratio
- **Blog images**: 800×500px or 16:10 ratio

## Notes
- JPEG or WebP format recommended for best performance
- Compress images before uploading (aim for < 300KB per image)
- The site will show a broken image icon if files are missing — add a `placeholder.jpg` to avoid this
